ASSET FOLDER GUIDE — Mohd Sheharyar Portfolio
================================================

/assets/resume.pdf
  → Place your resume PDF here.
  → The Download Resume button links to this exact path.

/assets/swordedge/
  → ss-01.jpg  — Combat system screenshot
  → ss-02.jpg  — Skill tree UI screenshot
  → ss-03.jpg  — Environment/world screenshot
  → ss-04.jpg  — Cinematic sequence screenshot
  → Recommended: 1920×1080 JPG, under 300KB each (use squoosh.app to compress)

/assets/vr/
  → clothing-preview.mp4   — Short 10–15s looping VR clothing demo
  → keyboard-preview.mp4   — VR keyboard demo
  → ui-preview.mp4         — World-space UI demo
  → Recommended: H.264 MP4, max 5MB each, loop point at end

/assets/projects/
  → key-to-escape.jpg      — Maze game screenshot
  → android-game.jpg       — Android game screenshot
  → cinematic.jpg          — Cinematic storytelling screenshot
  → Recommended: 1280×720 JPG, under 200KB each

YOUTUBE/VIMEO EMBED
  → In index.html, find the SwordEdge video section.
  → Uncomment the <iframe> block and replace YOUR_VIDEO_ID.
  → Format: https://www.youtube.com/embed/YOUR_VIDEO_ID?rel=0&showinfo=0

CONTACT LINKS
  → In index.html, search for "REPLACE" comments — there are 4 to update:
    1. Email address
    2. LinkedIn URL
    3. GitHub URL
    4. Resume download link (already set to assets/resume.pdf)
