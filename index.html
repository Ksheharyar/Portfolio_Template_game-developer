// DEVELOPMENT JOURNEY

The Journey

From first Blueprint node to shipping on Android

Dug into Unreal Engine fundamentals — level design, actor placement, Blueprint basics, and understanding the UE5 project structure. Built several small prototype levels to understand lighting, collision, and basic player movement.

Deep-dived into gameplay systems design. Built SwordEdge — an action RPG featuring combo-based combat, a modular skill system, and Sequencer-driven cinematic storytelling. First significant project to combine gameplay engineering with narrative design.

Expanded into XR development on Meta Quest. Built three distinct VR systems: a clothing try-on simulation, a predictive virtual keyboard, and a world-space UI interaction framework. Each project pushed the boundary of what natural spatial interaction can feel like inside a headset.

Currently in the final stretch of an Android game release. Focused on performance optimisation, touch input refinement, and Google Play integration. Actively seeking roles as a Gameplay Systems Developer or Technical Designer at game studios working in UE5 or XR.
